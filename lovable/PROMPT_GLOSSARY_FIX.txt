=== STEP 2 — Fix page load logic ===
On Glossary & Pathway page mount:

```
import glossaryStatic from "@/data/glossary_pathway.json";
import { fetchGlossaryPathway, isLiveDataEnabled } from "@/lib/agentApi";

// try live first, fall back to static import
let data = glossaryStatic;
if (isLiveDataEnabled()) {
  try {
    data = await fetchGlossaryPathway();
  } catch (e) {
    console.warn("Live glossary failed, using static", e);
  }
}
```
