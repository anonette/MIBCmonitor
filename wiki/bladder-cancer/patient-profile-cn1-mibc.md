# Patient Profile: cN1 cM0 Muscle-Invasive Urothelial Bladder Cancer

**Updated:** 2026-06-06

## Clinical Profile (Target for Evidence Matching)

| Feature | Value | Evidence-search implication |
|---------|-------|----------------------------|
| Histology | Muscle-invasive urothelial carcinoma, high grade | Standard MIBC trial eligibility |
| Pathology (TURBT) | pT2, L1 (lymphovascular invasion) | High-risk localized bladder; L1 not always reported in trial subgroups |
| Clinical nodes | cN1 — PET-positive external iliac node | **Key discriminator** — many foundational NAC trials excluded cN+ |
| Metastases | cM0 (no distant metastases) | Curative-intent perioperative setting, not metastatic maintenance |
| Treatment intent | Systemic therapy **before** possible radical cystectomy | Perioperative/neoadjuvant evidence prioritized over 1L metastatic |
| Geography / practice | Czech oncology context | Compare against EAU/NCCN cisplatin NAC standard and newer perioperative IO |

## What Counts as a "Match" in This Wiki

Trials are scored on how many enrolled patients resemble this profile:

**Tier A — Direct match (perioperative, cN1, cM0, bladder MIBC)**
- Clinical stage T2–T4, N1, M0
- Planned radical cystectomy after systemic therapy
- Example: EV-ECLIPSE (NCT05239624)

**Tier B — Partial nodal match**
- Node-positive allowed but minority of cohort (e.g., NIAGARA cN1 n=58; KEYNOTE-905 cN1 ~5%)
- Or lymph-node–only **metastatic** disease without planned curative surgery (EV-302 LN-only n=207)

**Tier C — Same histology/stage but wrong setting**
- Metastatic first-line (EV-302, EV-103, JAVELIN 100, CheckMate 901)
- Node-negative only (SWOG 8710, BA06)
- Post-surgery only (CheckMate 274, AMBASSADOR)

**Tier D — Retrospective / meta-analytic cN+ data**
- Not randomized EV+pembro evidence, but quantifies cN+ outcomes with cisplatin NAC

## Argument the Doctor Will Likely Frame

Czech standard: **cisplatin-based neoadjuvant chemotherapy → cystectomy** (SWOG 8710, BA06, meta-analyses).

Patient-specific question: whether **perioperative enfortumab vedotin + pembrolizumab** has enough evidence in **cN1 cM0** disease to justify off-label / non-reimbursed use vs approved alternatives (NIAGARA if cisplatin-fit; KEYNOTE-905 if cisplatin-ineligible).
